发表于 2020-03-12 分类于 解三角形

三角形问题的求解主要涉及到正弦定理、余弦定理和面积公式的运用.

正弦定理与余弦定理

正弦定理

在一个三角形中，各边和它对角的正弦值的比相等.

△ABC\triangle ABC△ABC，若角 A,B,CA,B,CA,B,C 所对的边分别是 a,b,ca,b,ca,b,c ，有

asin⁡A=bsin⁡B=Csin⁡C=2R\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{C}{\sin C} =2R sinAa​=sinBb​=sinCC​=2R

( RRR 为 △ABC\triangle ABC△ABC 外接圆的半径).

例题精讲

已知三角形一角及对边，求三角形面积、周长的最值

例1 (2014课标全国卷I, 理16) 有三角形的三个角 A,B,CA,B,CA,B,C 所对应的三个边分别为 a,b,ca,b,ca,b,c,其中已知 a=2a=2a=2,并且有 (2+b)(sin⁡A−sin⁡B)=(c−b)sin⁡C(2+b)(\sin A -\sin B)=(c-b)\sin C(2+b)(sinA−sinB)=(c−b)sinC,求△ABC\triangle ABC△ABC面积的最大值.

分析 问题等价于 a=2a=2a=2，A=π3A=\dfrac{\pi}{3}A=3π​，求 S△ABCS_{\triangle ABC}S△ABC​ 面积的最大值,显然面积公式应采用S△ABC=12bcsin⁡AS_{\triangle ABC}=\dfrac{1}{2}bc\sin AS△ABC​=21​bcsinA.

思路一 余弦定理角化边，不等式求最值

问题涉及三角形的三边和一角，可以借助余弦定理，转化为边，利用不等式求最值.

根据余弦定理 a2=b2+c2−2bccos⁡Aa^2=b^2+c^2-2bc\cos Aa2=b2+c2−2bccosA，代入数据，得

4=b2+c2−bc.4=b^2+c^2-bc. 4=b2+c2−bc.

根据重要不等式 b2+c2⩾2bcb^2+c^2\geqslant 2bcb2+c2⩾2bc 得

4=b2+c2−bc⩾2bc−bc=bc,4=b^2+c^2-bc \geqslant 2bc-bc=bc, 4=b2+c2−bc⩾2bc−bc=bc,

即

bc⩽4.bc \leqslant 4. bc⩽4.

所以

S△ABC=34bc⩽34×4=3.S_{\triangle ABC}=\dfrac{\sqrt{3}}{4}bc \leqslant \dfrac{\sqrt{3}}{4} \times 4=\sqrt{3}. S△ABC​=43​​bc⩽43​​×4=3​.

思路二 正弦定理边化角，三角函数求最值

已知条件为三角形的一边及其对角，可以借助正弦定理，转化为角，求三角函数的最值.

bsin⁡B=csin⁡C=asin⁡A=2sin⁡π3=433.\dfrac{b}{\sin B}=\dfrac{c}{\sin C}=\dfrac{a}{\sin A}=\dfrac{2}{\sin \dfrac{\pi}{3}}=\dfrac{4\sqrt{3}}{3}. sinBb​=sinCc​=sinAa​=sin3π​2​=343​​.

b=433sin⁡B,c=433sin⁡C.b=\dfrac{4\sqrt{3}}{3}\sin B,c=\dfrac{4\sqrt{3}}{3}\sin C. b=343​​sinB,c=343​​sinC.

因为A+B+C=πA+B+C=\piA+B+C=π,所以
