解三角形最值问题归纳

发表于 2020-03-12 分类于 解三角形

三角形问题的求解主要涉及到正弦定理、余弦定理和面积公式的运用.

正弦定理与余弦定理

正弦定理

在一个三角形中，各边和它对角的正弦值的比相等.

△ABC\triangle ABC△ABC，若角 A,B,CA,B,CA,B,C 所对的边分别是 a,b,ca,b,ca,b,c ，有

asin⁡A=bsin⁡B=Csin⁡C=2R\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{C}{\sin C} =2R sinAa​=sinBb​=sinCC​=2R

( RRR 为 △ABC\triangle ABC△ABC 外接圆的半径).
